How to Use URL Encoder & Decoder

  1. Enter the text or URL you want to encode or decode
  2. Select the appropriate encoding type
  3. Click "Encode" or "Decode" to process the text
  4. Copy the result or use the swap function to reverse the operation
Encoding Types Explained
URL Encoding

Converts special characters to percent-encoded format (%XX) for use in URLs.

HTML Encoding

Converts special characters to HTML entities (&, <, >, etc.).

Base64 Encoding

Converts text to Base64 encoded format for safe transmission.

Unicode Encoding

Converts characters to Unicode escape sequences (\uXXXX).
