How to Use Case Converter

  1. Enter your text in the input area
  2. Select the desired case type from the dropdown
  3. Click "Convert Text" or see real-time conversion
  4. Copy the converted text to your clipboard
Available Case Types
Basic Cases
  • UPPERCASE: ALL LETTERS IN CAPS
  • lowercase: all letters in small case
  • Title Case: First Letter Of Each Word Capitalized
  • Sentence case: First letter of sentence capitalized
Programming Cases
  • camelCase: firstWordLowerCase
  • PascalCase: FirstWordCapitalized
  • snake_case: words_separated_by_underscores
  • kebab-case: words-separated-by-hyphens
