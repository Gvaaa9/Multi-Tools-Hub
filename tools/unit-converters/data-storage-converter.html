Supported Data Storage Units
Common Units
  • Byte (B) - 8 bits
  • Kilobyte (KB) - 1,024 bytes
  • Megabyte (MB) - 1,024 KB
  • Gigabyte (GB) - 1,024 MB
  • Terabyte (TB) - 1,024 GB
Large Units
  • Petabyte (PB) - 1,024 TB
  • Exabyte (EB) - 1,024 PB
  • Zettabyte (ZB) - 1,024 EB
  • Yottabyte (YB) - 1,024 ZB
