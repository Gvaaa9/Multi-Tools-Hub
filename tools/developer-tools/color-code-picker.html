Color Formats Explained
HEX
  • Format: #RRGGBB
  • Range: #000000 to #FFFFFF
  • Used in: CSS, HTML
RGB
  • Format: rgb(r, g, b)
  • Range: 0-255 for each value
  • Used in: CSS, graphics
HSL
  • Format: hsl(h, s%, l%)
  • Hue: 0-360°, Saturation: 0-100%, Lightness: 0-100%
  • Used in: CSS, design
