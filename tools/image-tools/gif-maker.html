How to Create GIFs

  1. Upload multiple images by clicking "Choose Files" or dragging and dropping
  2. Arrange the images in the order you want them to appear in the GIF
  3. Adjust the frame delay, dimensions, and loop settings
  4. Click "Create GIF" to generate your animated GIF
  5. Download your GIF file
GIF Settings Explained
Frame Delay

Time each frame is displayed. Lower values = faster animation.

Dimensions

Width and height of the final GIF. Smaller = smaller file size.

Loop Count

How many times the GIF should loop. 0 = infinite loop.

Image Order

Images are processed in the order they were uploaded.
