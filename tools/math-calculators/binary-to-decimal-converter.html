Number Systems Explained
Binary (Base 2)
  • Uses digits 0 and 1
  • Used in computers
  • Example: 1010 = 10 in decimal
Hexadecimal (Base 16)
  • Uses digits 0-9 and A-F
  • Common in programming
  • Example: FF = 255 in decimal
